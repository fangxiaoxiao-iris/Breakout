1. The additional feature is that some bricks need several hits to be destroyed.

2. To run the program, use gradle build and gradle run --args='100 3', you can change the parameters.

3. The first parameter should be from 25-50, the second one should be 3 (slow), 4(medium) and 5(fast).

4. Please wait for a few seconds for the start page to show up. Press the play button to start the game.
